# TaxDibo

TaxDibo is a Vite + React + TypeScript dashboard starter with a shadcn-inspired interface and a responsive Holy Grail layout.

## Features

- Two-item side navigation for Home and Profile
- Top action bar with notifications, profile action, and theme toggle
- Responsive dashboard layout with glassmorphism-inspired cards
- Light and dark mode support via Tailwind variables

## Getting started

1. Install dependencies
   ```bash
   npm install
   ```
2. Start the development server
   ```bash
   npm run dev
   ```
3. Build for production
   ```bash
   npm run build
   ```

## Tech stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- shadcn-style UI primitives
